# DemoQA Automation Framework

## Tech Stack
- C#
- Selenium WebDriver
- SpecFlow (BDD)
- NUnit

## Framework Design
- Page Object Model
- Dynamic element locator strategy
- Explicit waits (no Thread.Sleep)
- Separation of concerns

## Folder Structure
- Pages
- StepDefinitions
- Features
- Utilities
- Hooks

## How to Run
1. Clone repo
2. Restore NuGet packages
3. Build solution
4. Run tests via Test Explorer
